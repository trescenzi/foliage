# Foliage

A vim colorscheme based on fall colors.

## Currently Supports

The following syntaxes are explicitly supported currently. More are in the works.

- Javascript (yajs)
- CSS (polyglot)
- HTML (polyglot)
- JSX (polyglot)

Built using [vim colortemplate v2](https://github.com/lifepillar/vim-colortemplate).
